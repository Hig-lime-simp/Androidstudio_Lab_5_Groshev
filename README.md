# Лабораторная работа №5: Jetpack Compose

## Цель работы
Освоить основы разработки пользовательского интерфейса на Jetpack Compose, работу с контейнерами Row и Column, познакомиться с ARGB-моделью цветов, компонентом Scaffold и принципами Material Design.

## Описание приложения
Приложение демонстрирует различные аспекты работы с Jetpack Compose:
- Работу с контейнерами Row и Column
- Посимвольный вывод строки
- Управление размерами и расположением элементов
- Использование ARGB-цветов
- Создание интерфейса с помощью Scaffold
- Комбинирование различных компоузабл-функций
